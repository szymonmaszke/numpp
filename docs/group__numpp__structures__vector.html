Detailed Description

Class representing one-dimensional numerical vector.

This class provides vector with it's basic functionality. Every operations should be done in compile time.

Warning
This implementation is obviously faster in runtime, but the compilation time may mitigate the positive effects.
Example:
\\Create vector
numpp::vector<double, 5> x{1,2,3,4,5}; //Implicit transposition equal to false
numpp::vector<double, 5, true> y{1,2,3,4,5}; //Transposed vector
double scalar = y*x //Returns scalar y^T * x
auto matrix = x*y //Returns matrix of x * y^T
std::cout << norm::euclidean(x) << std::endl; //prints euclidean norm of the vector

Warning
If you are looking for more compile-runtime balanced implementations check Eigen, Armadillo or other libraries
#include"numpp/structures/vector.hpp"
