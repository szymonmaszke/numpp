Detailed Description

This module provides the means for performing finite difference differentiation.

Calculates the derivative by using finite difference formulas For higher order derivatives check numpp_symbolic_differentiation Symbolic Differentiation

Warning
For automatically calculated epsilon type has to fullfil std::is_floating_point trait.
To specify your own type used in automatic epsilon overload, you have to specialize is_floating_point STL trait. Such type has to provide additional epsilon trait and few other operations.
Check source code for more info.
