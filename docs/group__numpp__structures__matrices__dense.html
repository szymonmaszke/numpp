Detailed Description

Module containg constexpr dense matrix implementation.

Warning
Compilation time of this matrix may mitigate any runtime speedup.
For dissertation of the problem see /ref praca

This matrix implementation uses heavy TMP tricks and C++17 functionality is required in order for it to work.

Warning
If you are looking for more compile-runtime balanced implementations check Eigen, Armadillo or other libraries

Example:

//Declaration of matrix
numpp::matrix::dense<double, 3, 3> mat{
0, 2, 3.15
4, 8, 12,
5., 7.7, 4.2
};
//Perform matrix multiplication
//Other operations have operator overloaded as well
auto new = mat*mat;
//Operation against scalar
auto divided = mat/3.14;
auto transposed = transpose(mat);
//Uses iterators, so you can print it this way:
for(const auto& row: mat){
for(const auto& elem: row)
std::cout << elem << " ";
std::cout << std::endl;
}
