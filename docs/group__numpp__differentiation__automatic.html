Detailed Description

This module provides the means for performing automatic differentiation.

Only forward mode as of now, backward propagation should be provided later.

Only first order partial differentiation supported.

For higher order derivatives check Symbolic Differentiation

Warning
Only arithmetic types are backed by the implementation. To specify your own type, you have to specialize is_arithmetic STL trait. Such type has to provide overload for addition and mathematical oprations like sin, cos etc. (constexpr versions for GCC compiler)
